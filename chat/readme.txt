项目应用JS类库：

	iscoll.js + touch.js + jquery.js + vue.js + vue-router.js

LOGIN页面： 单页应用设计    将 登陆 和 注册 整合在同一个页面

INDEX页面： 单页应用设计    分4个部分

					A、显示历史选择的人员列表     (包含好友通讯页面)
					B、显示所有人员列表     (包含好友通讯页面)
					C、学习天地 
					D、其他外链拓展

其他外链页面： 直接页面链接跳转


develop warning:

1、 注意 vue 的驼峰式命名规范

2、注意 IScroll 设置 scrollToElement 是配置的时间，如果时间太短，可能导致组件之间通信配合不在一起

3、需要修改内容：
		①、由于头部的按钮不是相同的，不能写为固定，应修改成按钮的显示可控制(配置参数设置显示或者隐藏)
